

<---------------------Number Systems----------------------->

1.Decimal Number System
    base = 10
    digits = 0-9

2.Binary Number System
    base = 2
    digits = 0-1      

3.Octal Number System
    base = 8
    digits = 0-7

4.Hexadecimal Number System
    base = 16
    digits = 0-9 and A-F

<----------------------------Number System Conversion---------------------------->

binary to decimal 
decimal to binary
binary to octal
octal to hexadecimal
and so on...........
